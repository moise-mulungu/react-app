## Getting Started

To get a local copy of this app you can use :
```
git clone  https://github.com/moise-mulungu/react-app
```
# Install

- Run :
1. `npm create-<app's name> .`
2. `cd <app's name>`,
3. `npm start`

# Usage

Run `npm start` to open the live server
